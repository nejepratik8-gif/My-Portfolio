# My Portfolio Website

This is my personal portfolio website project. I made this website while learning frontend web development.

In this website I added some information about me, my skills and some sample projects. I used basic HTML, CSS and a little JavaScript to make the website interactive.

## What is in this project

* Home section
* About me section
* Skills section
* Projects section
* Contact form

The navigation menu also has smooth scrolling using JavaScript.

## Technologies I used

* HTML
* CSS
* JavaScript

## How to run the project

Download or clone the repository and open the **index.html** file in your browser.

## Why I made this project

I made this project to practice building a simple website and understand how HTML, CSS and JavaScript work together.
